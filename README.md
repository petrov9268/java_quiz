# Java Quiz

Simple Java Quiz web application. Created with [Spring Framework][spr].
Uses H2 in-memory database.
For learning purposes.

## Features:
* view, create, edit lessons
* add comments

## Requirements:
* JDK 1.8 or later
* Maven 3.2 or later
